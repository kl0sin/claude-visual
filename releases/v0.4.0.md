# v0.4.0 — Agent Inspector & Auto-Update

## New Features

- **Update Banner** — The desktop app now checks GitHub Releases on startup and displays a dismissible banner when a newer version is available, with a direct download link.
- **Completed Agent Expansion** — Completed agents in the Agent Processes panel are now expandable: clicking a finished agent reveals a grouped tool-usage summary (tool name, call count, failure count, and total duration) for that agent's session.

## UX Improvements

- **Agent Tool Rows Extracted** — The inline tool-action rendering logic has been extracted into a dedicated `ToolActionRow` component, making active-agent tool rows cleaner and more consistent with the new history view.
- **Expanded Agent Tool Computation** — Tool actions for the currently expanded completed agent are now pre-computed alongside active agents, so the detail view renders instantly without a loading flash.
- **Font Rendering in Desktop App** — Fixed font display in the Tauri desktop build by correcting the `index.html` font reference and adjusting `src/index.css` font-face declarations.

## Internal / Maintenance

- **`stop_reason` Column Added** — A new `stop_reason TEXT` column has been added to the `sessions` table via a non-destructive `ALTER TABLE` migration, enabling richer session-end tracking.
- **`SessionSelector` Cleanup** — Minor layout and style tweaks to session tab rendering for better readability.
