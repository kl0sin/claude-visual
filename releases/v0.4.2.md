## Bug Fixes

- **Download button now works in Tauri desktop app**: The update banner's "Download" link used an `<a target="_blank">` tag which is silently ignored by Tauri's webview. Replaced with `shell.open()` from `@tauri-apps/plugin-shell` to correctly open the release page in the system browser.
- **Update checker version tracking**: `CURRENT_VERSION` in the update checker was not being updated during releases, causing the app to never detect newer versions. Added to the release checklist.

## Internal / Maintenance

- **Added `shell:allow-open` permission**: The Tauri capabilities config now includes the `shell:allow-open` permission required by the shell plugin to open external URLs.
- **New dependency `@tauri-apps/plugin-shell`**: Added the npm counterpart of the already-installed Rust `tauri-plugin-shell` crate for frontend access to shell commands.
