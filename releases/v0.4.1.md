## Bug Fixes

- **Sidecar-compatible hook installation**: Hook installation no longer relies on an external shell script (`install.sh`), which was inaccessible from compiled Tauri sidecar binaries. Hooks config is now embedded directly in the server code and written to `~/.claude/settings.json` programmatically.

## Internal / Maintenance

- **Simplified history module**: Removed unused `HOOKS_INSTALL_SCRIPT` constant and replaced `Bun.spawn` shell invocation with direct file operations (`mkdir`, `copyFile`, `Bun.write`) for hook installation, improving reliability and portability.
